# GPT Codex Apartment

A clean, simple, and well-structured HTML/CSS/JavaScript web application.

## Features

- **Clean HTML Structure**: Semantic HTML5 with proper document structure
- **Modern CSS**: Responsive design with CSS Grid and Flexbox
- **Interactive JavaScript**: Smooth scrolling, form handling, and scroll animations
- **Responsive Design**: Mobile-friendly layout that works on all devices
- **Accessibility**: Proper semantic markup and ARIA labels

## Project Structure

```
GPT-Codex-Apartment/
├── index.html      # Main HTML file
├── styles.css      # CSS styling
├── script.js       # JavaScript functionality
├── .gitignore      # Git ignore rules
└── README.md       # Project documentation
```

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/hgarg1/GPT-Codex-Apartment.git
   cd GPT-Codex-Apartment
   ```

2. Open the application:
   - Simply open `index.html` in your web browser
   - Or use a local server (e.g., with Python):
     ```bash
     python -m http.server 8000
     ```
   - Then navigate to `http://localhost:8000`

## Features Included

- **Navigation**: Smooth scrolling navigation menu
